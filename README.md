# RSA Encryption and Decryption System

## Description
This project implements the RSA algorithm without using cryptography libraries.

## Features
- User inputs prime numbers p and q
- RSA parameters calculated (n, φ, e, d)
- Encryption of text
- Saving encrypted text to file
- Reading encrypted text
- Decryption of text
- Factorization of n to retrieve private key

## How to Run
1. Open project in PyCharm
2. Run main.py
3. Enter p, q and text → Encrypt
4. Use n and e → Decrypt

## Example Values
p = 61  
q = 53
